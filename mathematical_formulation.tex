\section{Introduction}
This document formally defines the mathematical constraints and transformation pathways used to convert raw ARC physical grids into discrete, rule-based programmatic states using a Harmonic Slot Encoder, Vector Quantizer, and Broadcast Decoder.

\section{The Slot Compete Layer (Attention over Slots)}
In standard transformers, attention is evaluated over sequence or spatial dimensions. In Slot Attention, competition is enforced by evaluating the Softmax function directly across the $K$ slots. Let $Q \in \mathbb{R}^{K \times D}$ represent the $K$ slot queries, and $K, V \in \mathbb{R}^{N \times D}$ represent the $N$ spatial image patches. The attention mask $\alpha$ is defined as:

\begin{equation}
\alpha_{i,j} = \frac{\exp\left(\frac{Q_i \cdot K_j}{\sqrt{D}}\right)}{\sum_{k=1}^K \exp\left(\frac{Q_k \cdot K_j}{\sqrt{D}}\right)}
\end{equation}

By summing across $k=1 \dots K$ in the denominator, any mass claimed by slot $i$ mathematically deducts available mass from all other slots, forcing permutations of object grouping.

\section{The Vector Quantization Bottleneck (Logic Gate)}
Because neural representations default to continuous floats, we project them into a discrete $V$-state codebook to learn programmatic routines (e.g., categorical rule logic). Given a continuous slot vector $Z_{cont} \in \mathbb{R}^{D}$ and a codebook $E \in \mathbb{R}^{128 \times D}$, the quantization snaps to the nearest euclidean vector:

\begin{equation}
z_q = e_k \quad \text{where} \quad k = \arg\min_j || Z_{cont} - e_j ||_2^2
\end{equation}

Because the $\arg\min$ operation is fundamentally non-differentiable, gradient flow back to the encoder is restored using a \textbf{Straight-Through Estimator (STE)}:

\begin{equation}
Z_{final} = Z_{cont} + \text{stop\_gradient}(z_q - Z_{cont})
\end{equation}

\section{Spatial Broadcast Decoding}
To redraw the original image natively from the 1D slot encodings, the Decoder applies a positional embedding ($P_{pos}$) and physically repeats the 1x1 embedding across the original grid height ($H$) and width ($W$):

\begin{equation}
Z_{spatial}^{(k)} = \text{Repeat}(Z_{final}^{(k)}, [H, W]) + P_{pos}
\end{equation}

Two independent Multi-Layer Perceptrons yield a spatial mask $\alpha'_k$ and a color map $C_k$ for each slot $k$. The final grid summation employs a spatial softmax over all slots:

\begin{equation}
\hat{Y} = \sum_{k=1}^K \left( \text{Softmax}(\alpha'_k)_{dim=slots} \right) \odot C_k
\end{equation}

\section{Loss Interventions}
To prevent catastrophic representation collapse, three rigid loss functions direct the gradients.

\subsection{Foreground Focal Cross-Entropy ($\mathcal{L}_{CE}$)}
To prevent the ``Winner-Take-All'' spatial monopoly where a single slot commands the entire background, non-zero foreground shapes are weighted by an external penalty scalar $W_{fg} = 50.0$. Let $p_t$ denote the focal confidence scalar:

\begin{equation}
\mathcal{L}_{CE} = \mathbb{E}\left[ (1 - p_t)^2 \cdot (-\log(p_t)) \cdot \begin{cases} 50.0 & \text{if } Y_{true} > 0 \\ 1.0 & \text{if } Y_{true} = 0 \end{cases} \right]
\end{equation}

\subsection{VQ Commitment Loss ($\mathcal{L}_{VQ}$)}
To prevent codebook death, the codebook vectors $e$ must be dragged dynamically toward the incoming continuous feature states during the backpropagation rollout:

\begin{equation}
\mathcal{L}_{VQ} = || z_q - \text{stop\_gradient}(Z_{cont}) ||_2^2
\end{equation}

\subsection{VICReg Variance Margin ($\mathcal{L}_{VIC}$)}
To stop the slots from suffering from informational collapse, a strict margin $\gamma$ is upheld. The standard deviation across the $D$ dimensions of the current batch must remain above $\gamma$, otherwise a massive hinge penalty applies:

\begin{equation}
\mathcal{L}_{VIC} = \frac{1}{D} \sum_{d=1}^D \max\left(0, \gamma - \sqrt{\text{Var}(Z_{:, d}) + \epsilon}\right)
\end{equation}
